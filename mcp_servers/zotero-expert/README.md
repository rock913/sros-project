# Zotero Expert MCP Server

## Purpose
Manages local Zotero library integration and citation management.

## Features
- Read from local Zotero library
- Ensure citation key consistency
- Write AI-generated notes back to Zotero
- Manage bibliography generation

## Integration
This enhanced server connects to local Zotero installations to provide seamless citation management for the SROS system.

## Usage
The server maintains citation integrity and provides access to the user's existing research library.
