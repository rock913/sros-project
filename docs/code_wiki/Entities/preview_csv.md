# preview_csv

## 概要 (Description)
预览 CSV 文件，返回统计摘要（行数、列数、列名、数据类型、样本行、空值计数）。入参：file_path(str)。出参：Dict 包含 ok 布尔值和 summary 或 error。副作用：读取文件。
