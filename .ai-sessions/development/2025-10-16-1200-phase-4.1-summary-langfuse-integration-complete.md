### 3. LangFuse 集成模式

#### 模式 1: 单 Trace (无 Span)
**适用**: 独立操作，无子步骤

```python
from langfuse import Langfuse
langfuse = Langfuse()

trace = langfuse.trace(
    name="Document Diff Generation",
    input={"old_text_length": len(old_text), "new_text_length": len(new_text)},
    tags=["document", "diff"]
)
# ... 业务逻辑 ...
trace.update(output={"diff_count": len(diffs)})
```

**示例**: `document_utils.py` 中的 `generate_paragraph_diff` 和 `detect_conflict`

---

#### 模式 2: Trace + Span
**适用**: 复合操作，包含多个子步骤

```python
trace = langfuse.trace(
    name="HITL Query Approval",
    input={"session_id": session_id, "research_topic": topic, "queries": queries},
    tags=["hitl", "query_approval"]
)

# 子步骤: 创建 HITL 请求 (数据库操作)
span = trace.span(
    name="Create HITL Request",
    input={"decision_type": "query_approval", "prompt": prompt},
    tags=["hitl", "db"]
)
request_id = create_hitl_request(...)
span.end(output={"request_id": request_id})
```

**示例**: `hitl_nodes.py` 中的 3 个 HITL 节点, `graph.py` 中的报告生成

---

#### 模式 3: Trace + 多个 Span
**适用**: 长时间运行的会话，包含多个不同类型的事件

```python
trace = langfuse.trace(
    name="WebSocket Research Session",
    input={"thread_id": thread_id, "message_count": len(messages)},
    tags=["websocket", "streaming"]
)

# 事件 1: 文档更新
doc_span = trace.span(name="WebSocket Document Update", input={...}, tags=[...])
# ... 处理 ...
doc_span.end(output={...})

# 事件 2: HITL 请求
hitl_span = trace.span(name="WebSocket HITL Request", input={...}, tags=[...])
# ... 处理 ...
hitl_span.end(output={...})

# 最终更新 trace
trace.update(output={"status": "completed", "paper_count": 15})
```

**示例**: `app.py` 中的 WebSocket 端点

#### 查询 2: HITL 决策类型分布
```sql
SELECT 
    name,
    COUNT(*) as count,
    AVG(duration) as avg_duration_ms
FROM traces
WHERE name LIKE 'HITL%'
GROUP BY name
ORDER BY count DESC
```

---

#### 查询 3: LLM 生成性能统计
```sql
SELECT 
    PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY duration) as p50_ms,
    PERCENTILE_CONT(0.95) WITHIN GROUP (ORDER BY duration) as p95_ms,
    AVG(duration) as avg_ms,
    MAX(duration) as max_ms,
    COUNT(*) as total_calls
FROM spans
WHERE name = 'LLM Generate Report'
AND timestamp > NOW() - INTERVAL '7 days'
```
