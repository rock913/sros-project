## 2. Aider Configuration for Qwen Max
To enable Aider to work with Qwen Max via the OpenAI compatible endpoint, we need a specific configuration.

### .aider.conf.yml Template
This configuration defines two profiles: `architect` (read-only) and `builder` (read-write), though in our plan Copilot acts as the primary Architect. The `builder` profile is crucial for Aider.

```yaml
# .aider.conf.yml
model: openai/qwen-max
openai-api-base: https://dashscope.aliyuncs.com/compatible-mode/v1
# Qwen Max requires QWEN_API_KEY to be set in the environment as OPENAI_API_KEY
# or explicit environment variable mapping if Aider supports it, but standard way is OPENAI_API_BASE + OPENAI_API_KEY

map-tokens: 1024 # Qwen supports large context
cache-prompts: true
```
