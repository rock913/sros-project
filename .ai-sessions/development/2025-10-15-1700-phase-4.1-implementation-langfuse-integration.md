### 2. HITL 节点 Trace 集成

#### 文件: `backend/src/agent/hitl_nodes.py`

**全局导入**:
```python
from langfuse import Langfuse
langfuse = Langfuse()
```

#### 2.1 Query Approval Node (查询审批)

**Trace 入口**:
```python
trace = langfuse.trace(
    name="HITL Query Approval",
    input={
        "session_id": state.get("session_id"),
        "research_topic": state.get("research_topic"),
        "queries": state.get("search_queries", [])
    },
    tags=["hitl", "query_approval"]
)
```

**Span: 创建 HITL 请求**:
```python
span = trace.span(
    name="Create HITL Request",
    input={
        "decision_type": "query_approval",
        "prompt": f"AI已为研究主题「{research_topic}」生成以下查询，是否继续？",
        "options": ["approve", "reject", "modify"],
        "context": {"research_topic": research_topic, "queries": queries}
    },
    tags=["hitl", "db"]
)
...
span.end(output={"request_id": request_id})
```

**采集数据**:
- Input: session_id, research_topic, queries (查询列表)
- Output: request_id (HITL 请求 ID)
- Tags: hitl, query_approval, db

### 3. 报告生成节点 Trace 集成

#### 文件: `backend/src/agent/graph.py`

**函数**: `retrieve_and_synthesize_report`

**Trace 入口**:
```python
from langfuse import Langfuse
langfuse = Langfuse()
trace = langfuse.trace(
    name="Report Synthesis",
    input={
        "session_id": state.get("session_id"),
        "research_topic": get_research_topic(state["messages"]),
        "paper_count": len(state.get("literature_abstracts", []))
    },
    tags=["report", "synthesis"]
)
```

**Span: LLM 生成报告**:
```python
llm_span = trace.span(
    name="LLM Generate Report",
    input={
        "prompt": prompt[:500] + "..." if len(prompt) > 500 else prompt,
        "model": cfg.generation_model,
        "context_length": len(rag_context)
    },
    tags=["llm", "report"]
)

try:
    response = completion(...)
    report = response.choices[0].message.content
    llm_span.end(output={"report_length": len(report), "word_count": len(report.split())})
except ServiceUnavailableError as e:
    llm_span.end(output={"error": str(e)})
    trace.update(status="error", status_message=str(e))
```

**采集数据**:
- Input: session_id, research_topic, paper_count, prompt (截断), model, context_length
- Output: report_length, word_count (或 error)
- Tags: report, synthesis, llm

#### 4.2 ConflictDetector.detect_conflict

**Trace**:
```python
trace = langfuse.trace(
    name="Document Conflict Detection",
    input={
        "base_text_length": len(base_text),
        "user_text_length": len(user_text),
        "ai_text_length": len(ai_text)
    },
    tags=["document", "conflict"]
)
...
# 无冲突
trace.update(output={"is_conflict": False, "conflict_type": "none"})

# 或有冲突

    "is_conflict": True,
    "conflict_type": "overlapping",
    "overlapping_count": len(overlapping)
})
```

**采集数据**:
- Input: base_text_length, user_text_length, ai_text_length
- Output: is_conflict, conflict_type, overlapping_count (如有)
- Tags: document, conflict

## 性能影响评估 (Performance Impact)

### Trace 采集开销
- **网络延迟**: ~10-50ms (异步上传到 LangFuse)
- **内存占用**: ~1-5MB per trace (取决于 payload 大小)
- **CPU 占用**: 可忽略 (<1%)

### 优化措施
- LangFuse SDK 默认使用异步上传，不阻塞主流程
- Trace 数据在后台批量发送
- 可配置采样率 (sampling) 降低生产环境开销
