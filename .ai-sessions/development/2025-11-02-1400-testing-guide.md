# Analytics Dashboard 改进 - 测试指南

## 🎯 测试目标

验证新的"研究启动操作栏"功能，确保用户可以直接从 Analytics Dashboard 启动研究任务。

---

## 🚀 启动测试环境

### 1. 启动后端服务

```bash
# 确保后端服务正在运行
cd /mnt/data/hyf/gemini-fullstack-langgraph-quickstart
docker compose up -d langgraph-api
```

### 2. 启动扩展开发模式

1. 在 VS Code 中打开 `vscode-extension` 文件夹
2. 按 `F5` 启动 Extension Development Host
3. 在新窗口中，按 `Ctrl+Shift+P` 打开命令面板
4. 搜索并执行: `Auto Researcher: Show Analytics Dashboard`

---

## ✅ 测试用例

### 测试 1: 基本界面渲染

**预期结果**:
- [ ] 顶部显示"🚀 Start New Research"标题
- [ ] 输入框 placeholder 显示示例主题
- [ ] "Start Research"按钮初始状态为禁用（灰色）
- [ ] 字符计数器显示 "0 / 200"
- [ ] 如果有历史数据，显示"📝 Recent Topics ▼"按钮

---

### 测试 2: 输入验证

**步骤**:
1. 在输入框中输入 `AI` (2 字符)
2. 观察按钮状态和字符计数器

**预期结果**:
- [ ] 字符计数器显示 "2 / 200"
- [ ] "Start Research"按钮保持禁用状态

**步骤**:
3. 继续输入至 `AI in healthcare` (17 字符)

**预期结果**:
- [ ] 字符计数器显示 "17 / 200"
- [ ] "Start Research"按钮变为启用状态（蓝色）

**步骤**:
4. 继续输入至超过 180 字符

**预期结果**:
- [ ] 字符计数器变为警告颜色（橙色/黄色）
- [ ] 按钮仍然启用

**步骤**:
5. 继续输入至超过 200 字符（应该被阻止）

**预期结果**:
- [ ] 输入框阻止输入（maxlength 生效）
- [ ] 字符计数器显示 "200 / 200"，变为错误颜色（红色）

---

### 测试 3: 快捷键操作

**步骤**:
1. 输入 `Quantum computing` (17 字符)
2. 按 `Esc` 键

**预期结果**:
- [ ] 输入框清空
- [ ] 字符计数器重置为 "0 / 200"
- [ ] 按钮变回禁用状态

**步骤**:
3. 输入 `Machine learning` (16 字符)
4. 按 `Enter` 键

**预期结果**:
- [ ] 研究立即启动（无需点击按钮）
- [ ] Dashboard 自动关闭
- [ ] 弹出进度面板，标题为 "Research: Machine learning"
- [ ] 显示通知: "🚀 Starting research on: 'Machine learning'"

---

### 测试 4: Recent Topics 下拉菜单（仅当有历史数据时）

**前提条件**: 至少完成过 1 次研究

**步骤**:
1. 点击"📝 Recent Topics ▼"按钮

**预期结果**:
- [ ] 下拉菜单显示
- [ ] 显示最多 5 个最近的研究主题
- [ ] 每个主题可点击

**步骤**:
2. 点击其中一个主题

**预期结果**:
- [ ] 下拉菜单自动关闭
- [ ] 选中的主题填充到输入框
- [ ] 字符计数器更新
- [ ] "Start Research"按钮启用
- [ ] 输入框自动获得焦点（可立即编辑）

**步骤**:
3. 再次打开下拉菜单
4. 点击菜单外部区域

**预期结果**:
- [ ] 下拉菜单自动关闭
- [ ] 输入框内容不变

- [ ] 显示通知: "🚀 Starting research on: 'AI safety and alignment'"

4. 观察进度面板更新

**预期结果**:
- [ ] 进度条更新
- [ ] 显示当前步骤（如 "📚 Found X papers"）

### 测试 6: 空状态处理（无历史数据时）

- [ ] 没有"Recent Topics"按钮（因为无历史数据）
- [ ] Summary Cards 显示 0
- [ ] 图表显示"暂无数据"提示

**步骤**:
2. 输入主题并启动研究

**预期结果**:
- [ ] 研究正常启动
- [ ] 完成后，再次打开 Dashboard
- [ ] 统计数据更新，显示 1 个会话
