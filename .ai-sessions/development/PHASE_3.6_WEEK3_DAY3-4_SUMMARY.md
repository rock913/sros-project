## 📊 技术实现细节

### DocumentUpdate 接口
```typescript
export interface DocumentUpdate {
    type: 'document_update';
    session_id: string;
    node: string;
    action: 'insert' | 'modify' | 'delete' | 'unchanged';
    paragraph_index: number;
    content: string;
    old_content?: string;
    line_range: { start: number; end: number };
    rationale: string;
    timestamp: string;
}
```

### 工作流程
```
WebSocket 消息
    ↓
handleDocumentUpdate(update, documentUri)
    ↓
1. 打开/显示文档
2. 计算变更范围（VS Code Range）
3. 创建装饰（根据 action 类型）
4. 添加到 pendingChanges Map
5. 应用装饰到编辑器
6. 刷新 CodeLens
7. 显示通知（带 Accept/Reject 按钮）
    ↓
用户操作
    ↓
acceptChange() / rejectChange()
    ↓
1. 应用/拒绝编辑（vscode.WorkspaceEdit）
2. 从 pendingChanges 移除
3. 更新装饰
4. 刷新 CodeLens
5. 显示确认消息
```

### Range 计算逻辑
```typescript
private calculateRange(document, update): vscode.Range {
    // 后端发送 1-based 行号，VS Code 使用 0-based
    const startLine = Math.max(0, update.line_range.start - 1);
    const endLine = Math.min(document.lineCount - 1, update.line_range.end - 1);
    
    return new vscode.Range(
        new vscode.Position(startLine, 0),
        new vscode.Position(endLine, document.lineAt(endLine).text.length)
    );
}
```

### WorkspaceEdit 应用
```typescript
const edit = new vscode.WorkspaceEdit();

switch (update.action) {
    case 'insert':
        edit.insert(editor.document.uri, range.start, update.content + '\n\n');
        break;
    case 'modify':
        edit.replace(editor.document.uri, range, update.content);
        break;
    case 'delete':
        edit.delete(editor.document.uri, range);
        break;
}

await vscode.workspace.applyEdit(edit);
```
