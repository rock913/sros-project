```
TestPaperSelectionNode      3/3   ✓  论文数量判断、全选决策
TestReportRevisionNode      3/3   ✓  报告修改、审批/修改决策
```

#### 修复详情
**问题**:
```
psycopg2.errors.ForeignKeyViolation: insert or update on table "hitl_decisions" 
violates foreign key constraint "hitl_decisions_session_id_fkey"
DETAIL: Key (session_id)=(test-session-123) is not present in table "research_sessions".
```

**根本原因**:
- `setup_test_session()` 仅在 `if __name__ == "__main__"` 块中调用
- pytest 运行时不执行该块，导致测试会话记录缺失

**解决方案**:
```python
# backend/tests/test_hitl_nodes.py
import pytest

@pytest.fixture(scope="module", autouse=True)
def setup_database():
    """Pytest fixture to setup test database before all tests in this module"""
    setup_test_session()
    yield
```

**验证流程**:
1. 单测验证: `test_first_call_creates_hitl_request` → ✅ 通过
2. 模块验证: 全部 9 个测试 → ✅ 全部通过 (0.15s)
3. 全量验证: 总测试数 85→88 (+3)

#### 测试用例详解

**1. Query Approval Node (查询审批节点)**
```python
test_first_call_creates_hitl_request()  # 首次调用创建审批请求
test_approve_decision()                 # 通过决策流程

```

**2. Paper Selection Node (论文选择节点)**

