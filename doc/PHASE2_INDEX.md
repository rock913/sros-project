## 🎯 关键信息速查

### 项目目标
将 VS Code 扩展从 Mock 模式升级到真实后端 API 集成

### 开发周期
**3 周** (21 天 + 1 天验收)

### 里程碑
- **M1 (Week 1):** 能启动真实研究
- **M2 (Week 2):** 实时进度更新 + 历史数据
- **M3 (Week 3):** 完整交互功能

### 核心文件
- `vscode-extension/src/api.ts` - API 客户端
- `vscode-extension/src/extension.ts` - 主扩展逻辑
- `vscode-extension/src/websocket.ts` - WebSocket 客户端（新建）

### 关键 API
- `POST /threads` - 创建会话
- `POST /threads/{id}/runs/stream` - 启动研究
- `WS /agent/stream` - 实时更新

## 🚀 快速开始（5分钟）

```bash
# 1. 启动后端
cd /mnt/data/hyf/gemini-fullstack-langgraph-quickstart
docker compose -f docker-compose-dev.yml up -d

# 2. 验证服务
curl http://localhost:8121/ok
# 应该返回: {"ok":true}

# 3. 打开扩展开发
cd vscode-extension
npm install
code .

# 4. 按 F5 启动调试

# 5. 测试 Mock 模式

```
